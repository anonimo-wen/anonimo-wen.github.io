¿Cómo funciona VLSM?

VLSM (Variable Length Subnet Masking) permite crear subredes de diferentes tamaños dentro de una misma red principal. Esta calculadora:

  • Ordena los hosts de mayor a menor automáticamente
  • Calcula el prefijo (/n) necesario para cada subred
  • Asigna las direcciones de red, hosts y broadcast
  • Muestra si hay suficientes hosts disponibles
